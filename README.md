# FlappyDemo
Flappy Bird game using libGDX. Forked from [FlappyDemo](https://github.com/BrentAureli/FlappyDemo) by [BrentAureli ](https://github.com/BrentAureli).

## Desktop
Run this gradle task in the terminal
```
./gradlew :desktop:run
```
